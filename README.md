# Brain-Tumor-Classification-using-Machine-Learning-Models
This Jupyter Notebook presents a comprehensive analysis of brain tumor classification using MRI images. The project implements and compares four different algorithms (2 machine learning + 2 deep learning) to classify brain tumors into four categories: glioma, meningioma, pituitary tumor, and no tumor.
